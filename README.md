# XRSfM

## Quick Start

### Requirements
+ C++17
+ GCC 7.5+
+ CMake 3.15+
+ XRPrimer

#### Ubuntu:
 
```
git clone git@gitlab.bj.sensetime.com:openxrlab/xrsfm.git
cd xrsfm/
cmake -B build && cmake --build build -j8

```
